CONTENTS OF THIS FILE
---------------------

 * Introduction
 Provides a block with D8's dropdown element to switch sites active language
 (rather than the Language Switcher block that is an unordered list of links).

 * Requirements
 - language (core)
 - block (core)

 * Recommended modules
 None related.

 * Installation
 Install into modules folder.

 * Configuration
 /admin/config/regional/dropdown-language-switcher

 * Troubleshooting
 Have 2 or more languages active.
 /admin/config/regional/language

 * FAQ
 Block will 'appear' when more than one language is active. Otherwise,
 content is empty.
